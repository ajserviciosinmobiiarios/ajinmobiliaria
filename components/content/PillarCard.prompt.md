Numbered editorial feature block — used for the three pillars (estrategia, comunicación, confianza) and pain points.

```jsx
<PillarCard index="01" title="Estrategia a medida">
  Diseñamos un plan específico para tu inmueble.
</PillarCard>
```

Tones: `light`, `dark`. `divider` draws a hairline rule above each block for a column layout.
